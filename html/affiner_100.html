distance2d R Documentation

2D Euclidean distances

Description

distance2d() computes 2D Euclidean distances.

Usage

distance2d(x, y)

Arguments

x

Either a Coord2D or Line2D class object
y

Either a Coord2D or Line2D class object

Examples

p <- as_coord2d(x = 1:4, y = 1:4)
distance2d(p, as_coord2d(0, 0))
