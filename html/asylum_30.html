local_authority_resettlement R Documentation

Resettlement by local authority

Description

Refugees resettled by resettlement scheme and local authority.

Usage

local_authority_resettlement

Format

A data frame with 3,953 rows and 8 variables:

Date

The period in which the refugee arrived in the UK

Year

Year

Quarter

Quarter

UK Region

Region of the UK in which the refugee was initially resettled

LAD Code

Local Authority (district) code as supplied by ONS

Local Authority

The Local Authority in which the refugee was initially resettled

Resettlement Scheme

The scheme under which the refugee was resettled

Persons

The number of refugees resettled

...

Source

https://www.gov.uk/government/statistical-data-sets/asylum-and-resettlement-datasets
