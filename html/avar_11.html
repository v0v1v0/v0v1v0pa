av_dr R Documentation

Calculate Theoretical Allan Variance for Drift Process

Description

This function allows us to calculate the theoretical allan variance for drift process.

Usage

av_dr(delta, n)

Arguments

delta

A double value for the noise parameter \delta.
n

An integer value for the size of the cluster.

Value

A double indicating the theoretical allan variance for the drift process.

Note

This function is based on the calculation of the theoretical allan variance for drift process raised in "Allan Variance of Time Series Models for Measurement Data" by Nien Fan Zhang, 2008, Metrologia, 45(5): 549. This calculation is fundamental and necessary for the study in "A Study of the Allan Variance for Constant-Mean Non-Stationary Processes" by Xu et al., 2017, IEEE Signal Processing Letters, 24(8): 1257â€“1260.

Examples

av1 = av_dr(delta = 1, n = 5)
av2 = av_dr(delta = 2, n = 8)
