ax-series R Documentation

Add data to a chart

Description

Add data to a chart

Usage

ax_series(ax, ...)

ax_series2(ax, l)

Arguments

ax

An apexchart() htmlwidget object.
...

Lists containing data to plot, typically list with two items: name and data.
l

A list.

Value

An apexchart() htmlwidget object.

Examples


# One serie
apexchart() %>% 
  ax_series(list(
    name = "rnorm",
    data = rnorm(10)
  ))

# Two series
apexchart() %>% 
  ax_series(
    list(
      name = "rnorm 1",
      data = rnorm(10)
    ),
    list(
      name = "rnorm 2",
      data = rnorm(10)
    )
  )
