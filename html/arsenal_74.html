Arguments

x, y, object

An object of class "arsenal_table"
i, j

A vector to index x with: either names of variables, a numeric vector, or a logical vector of appropriate length. i indexes the x-variables, and j indexes the by-variables.
...

Other arguments (only used in print.summary.arsenal_table)
value

A list of new labels.
all, all.x, all.y

Logicals, denoting which terms to keep if not all are in common.
format

Passed to kable: the format for the table. The default here is "markdown". To use the default in kable, pass NULL. If x$text specifies LaTeX or HTML formatting, that format is used in the table.
escape

Passed to kable: should special characters be escaped when printed?
width, min.split

Passed to smart.split for formatting of the "term" column.
