get_bins R Documentation

Get Bins of Metadata

Description

Creates unique age/heights-bins for each record to speed up joining

Usage

get_bins(meta_criteria, param)

Arguments

meta_criteria

Metadata dataset

A metadata dataset that is the same as that in derive_params_growth_age()/derive_params_growth_height()
param

Character vector

Accepted values: "AGE"/"HEIGHT_LENGTH"

Value

list of breaks and labels for age/height bins

See Also

Other internal: admiralpeds-package, find_closest_bin(), set_bins()
