consumo R Documentation

Consumer cases

Description

Retrospective basis to use in book examples.

Usage

consumo

Format

a data frame with 1000 rows and 9 columns.

id_processo

case identifier number (appeal)

assunto

type of the case

comarca

municipality of the case

valor

value of cause

tipo_litigio

configuration of the parties to the dispute

dec_val

second instance decision

dec_unanime

unanimity

dec_date

decision date

tempo

case time in days

Source

https://abj.org.br/

Examples


summary(consumo)
