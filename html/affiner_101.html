distance3d R Documentation

3D Euclidean distances

Description

distance3d() computes 3D Euclidean distances.

Usage

distance3d(x, y)

Arguments

x

Either a Coord3D or Plane3D class object
y

Either a Coord3D or Plane3D class object

Examples

p <- as_coord3d(x = 1:4, y = 1:4, z = 1:4)
distance3d(p, as_coord3d("origin"))
