Glycoalkaloids R Documentation

Data Glycoalkaloids

Description

A measurement of the Glycoalkaloids by two methods: HPLC and spectrophotometer.

Usage

data(Glycoalkaloids)

Format

A data frame with 25 observations on the following 2 variables.

HPLC

a numeric vector

spectrophotometer

a numeric vector

Source

International Potato Center. CIP - Lima Peru.

References

International Potato Center. CIP - Lima Peru.

Examples

library(agricolae)
data(Glycoalkaloids)
str(Glycoalkaloids)
