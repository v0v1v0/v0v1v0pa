fixAlifs R Documentation

Standardize different hamzas on alif seats

Description

Standardize different hamzas on alif seats in a string.

Usage

fixAlifs(texts)

Arguments

texts

A string from which different alifs are standardized.

Value

fixAlifs returns a string with standardized alifs.

Author(s)

Rich Nielsen

Examples

## Create string with Arabic characters 
x <- '\u0622 \u0623 \u0675'

## Standardize Alifs
fixAlifs(x)
