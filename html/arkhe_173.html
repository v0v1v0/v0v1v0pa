Matrix Predicates

Description

  • is_square() checks if a matrix is square.

  • is_symmetric() checks if a matrix is symmetric.

Usage

is_square(x)

is_symmetric(x)

Arguments

x

A matrix to be tested.

Value

A logical scalar.
