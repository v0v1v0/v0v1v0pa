tags R Documentation

List tags

Description

List tags

Usage

tags(...)

tag(name, ...)

as.tag(x)

Arguments

...

Additional options passed down to GET
name

(character) Name of the tag
x

Object to coerce to a tag.

Details

tags gets all your tag, tag gets a tag by name

Value

Many tag objects in a list

Examples

## Not run: 
# get all your tags
tags()

# get a tag by name
tag("stuffthings")
tag("helloworld")

## End(Not run)
## Not run: 
tag_create("pluto")
as.tag('pluto')
as.tag(tag_create("howdyhoneighbor"))

## End(Not run)
