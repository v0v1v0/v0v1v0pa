Description

Check When an Item was Cached

Usage

argoFloatsWhenCached(name, debug = 0L)

Arguments

name

character value, naming the item.
debug

an integer, passed to argoFloatsDebug().

Value

A logical value indicating whether a cached value is available.

Author(s)
