applications R Documentation

Asylum applications

Description

Asylum applications raised, by nationality, age, sex, UASC, applicant type, and location of application.

Usage

applications

Format

A data frame with 69,576 rows and 11 variables:

Date

Date

Year

Year

Quarter

Quarter

Nationality

Nationality of people applying

Region

World region the nation is in

Age

Age group of applicants

Sex

Gender/sex of applicants

Applicant type

'Main applicant' or 'Dependant'

UASC

Are they an unaccompanied asylum-seeking child?

Location of application

'At Port' or 'In Country'

Applications

Number of applications

...

https://www.gov.uk/government/statistical-data-sets/asylum-and-resettlement-datasets
