Shorten an object of class gaze

Description

Shorten an object of class gaze

Usage

shorten(x, xname = NULL, ref = 1)

Arguments

x

an object of class gaze
xname

A variable name
ref

Numeric Th number to be used as reference

Value

An object of class "gaze" which is described in gaze

Examples

data(acs,package="moonBook")
x=gaze(sex~.,data=acs)
shorten(x)
