get_all_subsets R Documentation

Get the items/names/ids of subsets from a named list

Description

Get the items/names/ids of subsets from a named list

Usage

get_all_subsets(list, name_separator = "/")

Arguments

list

a named list
name_separator

default is /

Value

a tibble

Examples

list = list(A = sample(LETTERS, 20),
            B = sample(LETTERS, 22),
            C = sample(LETTERS, 24),
            D = sample(LETTERS, 30, replace = TRUE))
get_all_subsets(list)
