resample_30hz

Description

Resample the raw data.

Usage

resample_30hz(raw = c(), sf = 30, verbose = FALSE)

Arguments

raw

Matrix containing raw data
sf

Sample frequency of raw data (Hz)
verbose

Print diagnostic messages

Value

resampled_data
