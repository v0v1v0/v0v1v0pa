kurtosis R Documentation

Finding the Kurtosis coefficient

Description

It obtains the value of the kurtosis for a normally distributed variable. The result is similar to SAS.

Usage

kurtosis(x)

Arguments

x

a numeric vector

Value

x

The kurtosis of x

See Also

skewness

Examples

library(agricolae)
x<-c(3,4,5,2,3,4,5,6,4,NA,7)
kurtosis(x)
# value is -0.1517996
