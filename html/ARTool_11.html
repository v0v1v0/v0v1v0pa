Higgins1990Table5.art R Documentation

Aligned Rank Transformed Version of Higgins1990Table5

Description

The ART version of Higgins1990Table5 as produced by the original ARTool, used to test the correctness of art output.

Format

A data frame with 48 observations on the following 10 variables.

Tray

a factor with levels "t1" .. "t12"

Moisture

a factor with levels "m1" .. "m4"

Fertilizer

a factor with levels "f1" .. "f4"

DryMatter

a numeric vector

aligned.DryMatter..for.Moisture

a numeric vector

aligned.DryMatter..for.Fertilizer

a numeric vector

aligned.DryMatter..for.Moisture.Fertilizer

a numeric vector

ART.DryMatter..for.Moisture

a numeric vector

ART.DryMatter..for.Fertilizer

a numeric vector

ART.DryMatter..for.Moisture.Fertilizer

a numeric vector

Source

Wobbrock, J. O., Findlater, L., Gergle, D., and Higgins, J. J. ARTool. https://depts.washington.edu/acelab/proj/art/.
