show-methods R Documentation

Print Functions for Objects of the Package 'sp' Used in 'adehabitatMA'

Description

These are functions allowing a shorter print of the content of objects of class SpatialPolygonsDataFrame , SpatialPixelsDataFrame, SpatialPixels, and SpatialGridDataFrame. Original printing methods can be recovered by setting adeoptions(shortprint=FALSE).
