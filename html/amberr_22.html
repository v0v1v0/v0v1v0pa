Description

Get the groups

Usage

amber.groups(amber, query = list(), skip = 0, limit = 100, df = TRUE)

Arguments

amber

An Amber object
query

The search query
skip

Number of items to skip
limit

Max number of items
df

Return a data.frame (default is TRUE)

Value

A data.frame (or a named list of raw results when 'df' is FALSE)
