frac.ranks R Documentation

Fractional Ranks

Description

This function computes fractional ranks which are required for the S-Gini coefficient.

Usage

frac.ranks(x, w = NULL)

Arguments

x

a vector with sorted income values.

w

a vector of weights.

Value

returns the fractional ranks.

Author(s)

Alexander Sohn

References
