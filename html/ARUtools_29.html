example_clean R Documentation

Example cleaned recording meta data

Description

A data frame with examples of correctly formatted metadata with added site-level information

Usage

example_clean

Format

example_clean

A data frame with 42 rows and 10 columns:

file_name

Name of the file

type

File type

path

Relative file path including file name

aru_type

ARU model

aru_id

ARU ids

site_id

Site ids

date_time

Recording date/time

date

Recording date

longitude

Latitude in decimal degrees

latitude

Longitude in decimal degrees
