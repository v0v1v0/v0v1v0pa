lagm R Documentation

Lag a matrix

Description

Take the lag of a variable using the specified number of lag(s).

Usage

lagm(x, k)

Arguments

x

a matrix object

k

number of lags

Value

x

dimentional matrix with k

Examples

x <- cbind(x1 = c(23, 45, 56, 78, 98, 21, 17),
            x2 = c(3, 4, 6, 7, 2, 9, 17))
lagm(x, k = 3)
