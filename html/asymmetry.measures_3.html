Epanechnikov kernel

Description

Implementation of the Epanechnikov kernel.

Usage

  Epanechnikov(x)

Arguments

x

A vector of data points between -\sqrt{5} and \sqrt{5} where the kernel will be evaluated.

Details

Implements:

K(u)= \frac{3}{4\sqrt{5}} \left (1-\frac{x^2}{5} \right )

for |x| \le \sqrt{5}

Value

The value of the kernel at x
