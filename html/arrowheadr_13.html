nudger R Documentation

Nudge columns of a matrix by fixed amounts

Description

Nudge columns of a matrix by fixed amounts

Usage

nudger(x, nudge)

Arguments

x

a matrix
nudge

a single value or a vector with length equal to the number of columns in x

Value

matrix

Examples

nudger(matrix(0, nrow = 2, ncol = 2), nudge = c(0,1))
