cal_names R Documentation

Calendar names

Description

cal_names() returns the names of the holidays in a calendar.

Usage

cal_names(x)

Arguments

x

⁠[rcalendar]⁠

An rcalendar.

Value

A character vector of holiday names.

Examples

x <- rcalendar(hol_christmas(), hol_new_years_day())
cal_names(x)
