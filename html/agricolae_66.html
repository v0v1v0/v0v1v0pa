lastC R Documentation

Setting the last character of a chain

Description

A special function for the group of treatments in the multiple comparison tests. Use plot.group.

Usage

lastC(x)

Arguments

x

letters

Value

x

Returns the last character of a string

Author(s)

Felipe de Mendiburu

See Also

plot.group

Examples

library(agricolae)
x<-c("a","ab","b","c","cd")
lastC(x)
# "a" "b" "b" "c" "d"
