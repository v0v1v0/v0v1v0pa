support_applications R Documentation

Applications for section 95 support

Description

Applications for section 95 support, by nationality, support type granted, and group type.

Usage

support_applications

Format

A data frame with 4,756 rows and 6 variables:

Year

Year

Nationality

Nationality of people receiving support

Region

World region the nation is in

Support type granted

'Accommodation', 'Closed / Invalid', or 'Subsistence Only'

Group type

'Single adult', 'Family', or 'Unknown'

Applications

Number of applications

...

Source

https://www.gov.uk/government/statistical-data-sets/asylum-and-resettlement-datasets
