Description

Extract station data from API object

Usage

stations(x)

Arguments

x

An nrel_api object.

Value

A data frame with the station listing from the nrel_api object.

Examples

my_stations <- get_all_stations(MY_API_KEY, my_params)
stations(my_stations)
