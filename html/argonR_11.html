argonContact R Documentation

Create a Boostrap 4 argon contact form

Description

Build an argon contact form

Usage

argonContact(
  title = NULL,
  subtitle = NULL,
  name = "Your name",
  mail = "Email Adress",
  subject = "Subject",
  message,
  mailto,
  width = 8
)

Arguments

title

Form title.
subtitle

Form subtitle.
name

Contact name.
mail

Contact mail.
subject

Message subject.
message

For message
mailto

Adress to which send the form.
width

Form width. Between 1 and 12.

Note

Need to have an active formspree.io account. See https://formspree.io. Only works on web-server!
