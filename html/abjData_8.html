pnud_siglas R Documentation

UNDP Acronyms

Description

A dataset that serves as a glossary of available variables.

Usage

pnud_siglas

Format

a data frame with 236 rows e 4 columns:

sigla

all acronyms available

nome_curto

short name of the variable

nome_longo

long name of the variable

definicao

definition of acronym

Source

https://www.br.undp.org/content/brazil/pt/home/idh0/atlas-do-desenvolvimento-humano/atlas-dos-municipios.html

Examples


summary(pnud_siglas)
