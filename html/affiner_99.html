distance1d R Documentation

1D Euclidean distances

Description

distance1d() computes 1D Euclidean distances.

Usage

distance1d(x, y)

Arguments

x

Either a Coord1D or Point1D class object
y

Either a Coord1D or Point1D class object

Examples

p <- as_coord1d(x = 1:4)
distance1d(p, as_coord1d(0))
