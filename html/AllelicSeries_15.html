Generate Genotype Matrix

Description

Generate Genotype Matrix

Usage

GenGenoMat(n, snps, maf_range = c(0.005, 0.01))

Arguments

n

Sample size.
snps

Number of SNP in the gene.
maf_range

Range of minor allele frequencies: c(MIN, MAX).

Value

(n x snps) numeric matrix.
