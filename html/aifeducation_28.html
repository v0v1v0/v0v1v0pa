is.null_or_na R Documentation

Check if NULL or NA

Description

Function for checking if an object is NULL or NA

Usage

is.null_or_na(object)

Arguments

object

An object to test.

Value

Returns FALSE if the object is not NULL and not NA. Returns TRUE in all other cases.
