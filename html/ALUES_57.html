CITRUSWater R Documentation

Citrus water requirement for land evaluation

Description

A dataset containing the water characteristics of the crop requirements for farming Citrus.

Format

A data frame with 5 rows and 8 columns

Details

The following are the factors for evaluation:

  • WyAv - Annual precipitation (mm)

  • WmDryLen - Length dry season (months : P < 1/2 PET)

  • WmhColdXm - Relative humidity of coldest month if frost (%)

  • WmhAv4 - Relative humidity at harvest stage (%)

  • WmhAv3 - Relative humidity of maturation Stage (%)
