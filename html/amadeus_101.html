process_sedac_groads R Documentation

Process roads data

Description

The process_sedac_groads() function imports and cleans raw road data, returning a single SpatVector object.

Usage

process_sedac_groads(path = NULL, extent = NULL, ...)

Arguments

path

character(1). Path to geodatabase or shapefiles.
extent

numeric(4) or SpatExtent giving the extent of the raster if NULL (default), the entire raster is loaded
...

Placeholders.

Value

a SpatVector object

Note

U.S. context. The returned SpatVector object contains a ⁠$description⁠ column to represent the temporal range covered by the dataset. For more information, see https://sedac.ciesin.columbia.edu/data/set/groads-global-roads-open-access-v1/metadata.
