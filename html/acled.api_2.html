get.api.regions R Documentation

ACLED API region support function

Description

List of region names and codes as they are supplied by the ACLED API user guide.

Usage

get.api.regions()

Details

This single-purpose support function does not take arguments. It is set up as a convenient way to return region names and region codes to be used in the acled.api() main function's region argument.

Value

A list of length 2.

  1. A data frame object containing ACLED region names and codes.

  2. A string with version information.
