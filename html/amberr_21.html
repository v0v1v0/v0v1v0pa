amber.group R Documentation

Get a group

Description

Get a user by name or identifier.

Usage

amber.group(amber, id)

Arguments

amber

An Amber object
id

Group's name or identifier

Value

A group object as a named list

See Also

Other users functions: amber.groups(), amber.user(), amber.users()

Examples

## Not run: 
a <- amber.login("https://amber-demo.obiba.org")
amber.group(a, id = "obiba.org")
amber.group(a, id = "615abb061799dd6a2cdbbb10")
amber.logout(a)

## End(Not run)
