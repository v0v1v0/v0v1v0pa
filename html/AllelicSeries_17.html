OLS R Documentation

Ordinary Least Squares

Description

Fits the standard OLS model.

Usage

OLS(y, X)

Arguments

y

(n x 1) Numeric vector.
X

(n x p) Numeric matrix.

Value

List containing the following:

  • BetaRegression coefficient.

  • VOutcome variance.

  • SEStandard errors.

  • ZZ-scores.
