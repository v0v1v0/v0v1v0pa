unemp R Documentation

U.S. Unemployment

Description

Monthly U.S. Unemployment series (1948-1978, n = 372)

Usage

data(unemp)

Format

The format is: Time-Series [1:372] from 1948 to 1979: 235 281 265 241 201 ...
