MGSM27 R Documentation

Myeloma Genome Signature Matrix 27

Description

Newman et al's 2015 plus 5 myeloma specific cell types. Osteoclasts, Adipocytes, Osteoblasts, Multiple Myeloma Plasma Cells, and Plasma Memory Cells

Usage

data("MGSM27")
