fit2lik R Documentation

extract likelihood information with a coxph object

Description

extract likelihood information with a coxph object

Usage

fit2lik(x)

Arguments

x

An object of class "coxph" or "survreg"

Value

A string

Examples

library(survival)
fit=coxph(Surv(time,status) ~rx,data=anderson)
fit2lik(fit)
