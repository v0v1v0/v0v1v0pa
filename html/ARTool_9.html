Higgins1990Table1.art R Documentation

Aligned Rank Transformed Version of Higgins1990Table1

Description

The ART version of Higgins1990Table1 as produced by the original ARTool, used to test the correctness of art output.

Format

A data frame with 36 observations on the following 10 variables.

Subject

a factor with levels "s1" .. "s36"

Row

a factor with levels "r1" .. "r3"

Column

a factor with levels "c1" .. "c3"

Response

a numeric vector

aligned.Response..for.Row

a numeric vector

aligned.Response..for.Column

a numeric vector

aligned.Response..for.Row.Column

a numeric vector

ART.Response..for.Row

a numeric vector

ART.Response..for.Column

a numeric vector

ART.Response..for.Row.Column

a numeric vector

Source

Wobbrock, J. O., Findlater, L., Gergle, D., and Higgins, J. J. ARTool. https://depts.washington.edu/acelab/proj/art/.
