archivist.github-package R Documentation

Tools for Archiving, Managing and Sharing R Objects via GitHub

Description

GitHub integration extension of the archivist package - tool for storing, restoring and searching for R objects. More about archivis.github can be found on marcinkosinski.github.io/archivist.github/

Details

More about archivis.github can be found on marcinkosinski.github.io/archivist.github/ and about archivist in posts' history on https://pbiecek.github.io/archivist/articles/posts.html
