myformat R Documentation

Convert data.frame to printable format

Description

Convert data.frame to printable format

Usage

myformat(x, showid = FALSE, digits = 3)

Arguments

x

A data.frame
showid

logical if TRUE, show id
digits

Integer indicating the number of decimal places

Value

A data.frame

Examples

fit=lm(mpg~wt*hp,data=mtcars)
gaze(fit) %>% myformat()
