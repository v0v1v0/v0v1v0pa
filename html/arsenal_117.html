%nin% R Documentation

Not in

Description

The not-in operator for R.

Usage

x %nin% table

Arguments

x

vector or NULL: the values to be matched.
table

vector or NULL: the values to be matched against.

Value

The negation of %in%.

Author(s)

Raymond Moore

See Also

%in%

Examples

1 %nin% 2:10
c("a", "b") %nin% c("a", "c", "d")
