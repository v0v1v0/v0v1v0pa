Line2D R Documentation

2D lines R6 Class

Description

Line2D is an R6::R6Class() object representing two-dimensional lines.

Public fields

a

Numeric vector that parameterizes the line via the equation a * x + b * y + c = 0.

b

Numeric vector that parameterizes the line via the equation a * x + b * y + c = 0.

c

Numeric vector that parameterizes the line via the equation a * x + b * y + c = 0.

Examples
