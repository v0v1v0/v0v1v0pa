resample_10hz

Description

Get data back to 10 Hz for accumulation

Usage

resample_10hz(trim_data = c(), verbose = FALSE)

Arguments

trim_data

Matrix containing the trimmed/thresholded data
verbose

Print diagnostic messages

Value

Resampled data
