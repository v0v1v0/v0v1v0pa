argonPage R Documentation

Create a Boostrap 4 argon page

Description

Build an argon page

Usage

argonPage(
  ...,
  title = NULL,
  description = NULL,
  author = NULL,
  navbar = NULL,
  footer = NULL,
  favicon = NULL,
  analytics = NULL
)

Arguments

...

Body content
title

App title.
description

Purpose.
author

Author.
navbar

Navbar.
footer

Footer.
favicon

Website favicon. The png must be located in inst/images.
analytics

Website analytics such as Hotjar or google analytics. Must be wrapped in tagList or list. Moreover the script must be contained in a <script></script> tag: if it is not already the case, use tags$script.

Author(s)

David Granjon, dgranjon@ymail.com

Examples

if(interactive()){
 library(argonR)
 argonPage(
  title = "ArgonR Static Template",
  author =  "Somebody",
  description = "HTML Static Template",
  navbar = argonNavbar(id = "navbar"),
  footer = argonFooter(),
  # main content
  argonSection(),
  argonSection(),
  argonSection(),
  argonSection(),
  argonSection()
 )
}
