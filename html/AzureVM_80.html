Details

Some predefined network security rule objects, to unblock commonly used ports.

  • HTTP: TCP port 80

  • HTTPS: TCP port 443

  • JupyterHub: TCP port 8000

  • RDP: TCP port 3389

  • RStudio Server: TCP port 8787

  • SSH: TCP port 22

  • SQL Server: TCP port 1433

  • SQL Server browser service: TCP port 1434
