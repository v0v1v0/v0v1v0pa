amber.user R Documentation

Get a user

Description

Get a user by email or identifier.

Usage

amber.user(amber, id)

Arguments

amber

An Amber object
id

User's email or identifier

Value

A user object as a named list

See Also

Other users functions: amber.group(), amber.groups(), amber.users()

Examples

## Not run: 
a <- amber.login("https://amber-demo.obiba.org")
amber.user(a, id = "jim@muse.com")
amber.user(a, id = "6151b512268f582926d37f90")
amber.logout(a)

## End(Not run)
