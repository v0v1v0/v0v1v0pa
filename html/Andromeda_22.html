names.tbl_Andromeda R Documentation

Get the column names of an Andromeda table

Description

Get the column names of an Andromeda table

Usage

## S3 method for class 'tbl_Andromeda'
names(x)

Arguments

x

An table in an Andromeda object

Value

A character vector of column names

Examples

andr <- andromeda(cars = cars)
names(andr$cars)
# [1] "speed" "dist"
close(andr)
