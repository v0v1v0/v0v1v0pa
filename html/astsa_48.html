gdp R Documentation

Quarterly U.S. GDP

Description

Seasonally adjusted quarterly U.S. GDP from 1947(1) to 2018(3).

Format

The format is: Time-Series [1:287] from 1947 to 2018: 2033 2028 2023 2055 2086 ...

Source

https://tradingeconomics.com/united-states/gdp
