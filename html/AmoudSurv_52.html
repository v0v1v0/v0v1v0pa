Lognormal (LN) Survivor Hazard Function.

Usage

Arguments

: positive argument
kappa

: meanlog parameter
alpha

: sdlog parameter

the value of the LN Survivor function

Author(s)

Abdisalam Hassan Muse, Samuel Mwalili, Oscar Ngesa, Christophe Chesneau abdisalam.hassan@amoud.edu.so
