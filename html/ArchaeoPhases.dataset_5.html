KAPhasesChronoModel R Documentation

Ksar Akil phases calibrated by ChronoModel

Description

A data set

Usage

KAPhasesChronoModel

Format

A data frame with 30,000 rows and 9 variables:

iter

iteration of the MCMC algorithm

EPI.alpha

start date of EPI

EPI.beta

end date of EPI

UP.alpha

start date of UP

UP.beta

end date of UP

Ahmarian.alpha

start date of Ahmarian

Ahmarian.beta

end date of Ahmarian

IUP.alpha

start date of IUP

IUP.beta

end date of IUP
