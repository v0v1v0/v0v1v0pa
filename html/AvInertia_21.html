Kroneckerdelta function

Description

Kroneckerdelta function

Usage

kronecker_delta(i, j)

Arguments

i

a scalar index (usually row of a matrix)
j

a scalar index (usually column of a matrix)

Value

a scalar value. Returns 1 if i and j are equal otherwise returns 0

Author(s)

Christina Harvey

Examples


library(AvInertia)

# should return 1
kronecker_delta(1,1)

# should return 0
kronecker_delta(0,1)
