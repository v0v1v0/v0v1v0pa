aldatata.api_key R Documentation

Set ALTADATA API key

Description

Set ALTADATA API key

Usage

aldatata.api_key(api_key)

Arguments

api_key

Optionally passed parameter to set Altadata api_key.

Value

Returns invisibly the currently set api_key.

Examples

## Not run: 
  aldatata.api_key('foobar')

## End(Not run)
